Selectors

Selectors are fundamental in CSS as they determine which HTML elements the styles apply to.

Universal Selector

This selector applies to all elements of any type, useful for applying a general style across your webpage. 


            /* Universal selector example */
            * {
                font-family: Arial, sans-serif;
                color: black;
            }

Type Selectors

Type selectors target all instances of a specified HTML element based on their tag names. 


            /* Type selector example */
            h1 {
                font-size: 32px;
                color: darkblue;
            }

Class Selectors

Class selectors are used to apply styles to multiple elements that share the same class attribute.

This is a highlighted paragraph using a class selector. 

            /* Class selector example */
            .highlight {
                background-color: yellow;
                font-weight: bold;
            }

ID Selectors

ID selectors target specific elements by their unique ID attribute.

Welcome to My Website!

            /* ID selector example */
            #main-title {
                font-size: 36px;
                color: green;
            }

Grouping Selector

The grouping selector allows you to apply the same styles to multiple selectors at once. 


            /* Grouping selector example */
            h2, h3 {
                color: gray;
                margin-bottom: 10px;
            }

Chaining Selectors

Chaining selectors allows you to combine multiple selectors to apply styles only to elements that match all specified criteria.

This is an active box.

            /* Chaining selector example */
            .box.active {
                border: 2px solid red;
            }

Descendant Combinator

Combinators allow you to define relationships between selectors. The descendant combinator selects elements nested within another specified element.

This is a nested paragraph.


            /* Descendant combinator example */
            .container .inner p {
                color: blue;
            }
