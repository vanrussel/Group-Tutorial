Lesson 1: Intro to CSS

Introduction to CSS

Previously, you learned how HTML structures a web page. Now, let’s make it look appealing with CSS.

Lesson Overview

This lesson covers essential topics that will equip you with the skills to style your web pages effectively.

Basic Syntax

At its core, CSS is composed of rules that dictate how HTML elements are displayed on the web page. Each rule consists of a selector and a list of declarations. 


            /* Basic CSS syntax example */
            h1 {
                color: blue;
                font-size: 24px;
            }

Essential CSS Properties

Below are some essential CSS properties that will help you start styling web pages effectively.

Color and Background

The color property sets the color of text, and background-color defines the background color of elements. 


            /* Color and background example */
            body {
                background-color: #f0f0f0;
            }
            p {
                color: #333;
            }

Typography

CSS provides several properties to control font styles like font-family, font-size, and text-align. 


            /* Typography example */
            h1 {
                font-family: 'Arial', sans-serif;
                font-size: 24px;
                text-align: center;
            }
            p {
                font-size: 16px;
                line-height: 1.5;
            }

Image Dimensions

CSS allows you to control image dimensions with properties like width and height. 


            /* Image dimensions example */
            img {
                width: 100%;
                height: auto;
            }

Adding CSS to HTML

CSS can be added to HTML through external, internal, and inline methods. In this example, we’re using internal CSS.

External CSS

External CSS is the most common method, where you link a separate CSS file within the <head> section of your HTML. 


            
            <link rel="stylesheet" href="styles.css">

Internal CSS

Internal CSS is applied within the <style> element in the HTML <head> section, as shown in this document.

Inline CSS

Inline CSS applies styles directly to HTML elements using the style attribute. 


            <p style="color: red; font-size: 18px;">This text is red and larger!</p>

Conclusion

By mastering these CSS basics, you will be able to create visually stunning web pages that are engaging and user-friendly. Experiment with different properties and selectors to enhance your designs!
