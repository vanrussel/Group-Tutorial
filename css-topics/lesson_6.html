Lesson 6: Block & Inline

Introduction

CSS (Cascading Style Sheets) is essential for controlling the layout of HTML elements on a webpage. One of the key concepts in CSS is understanding the difference between block and inline elements. This distinction affects how elements are displayed, how they interact with each other, and how they contribute to the overall layout of a webpage.

Lesson Overview

This lesson will cover the following topics:

  • Definition of Block and Inline elements.
  • Characteristics of Block elements.
  • Characteristics of Inline elements.
  • Differences between Block and Inline elements.
  • Examples and use cases.

Block Elements

Block elements are HTML elements that occupy the full width available, with a new line before and after them. They create a "block" of space on the page and are typically used for larger sections of content. Common block elements include:

  • <div>
  • <p>
  • <h1> to <h6>
  • <ul>, <ol>, <li>
  • <header>, <footer>, <section>, <article>

Characteristics of block elements include:

  • Always start on a new line.
  • Take up the full width of their parent container by default.
  • Can have width and height set via CSS.
Example: 
div {
        width: 100%; /* Takes full width */
        background-color: lightblue;
        padding: 10px;
    }

Inline Elements

Inline elements, on the other hand, only take up as much width as necessary, and they do not start on a new line. They are typically used for smaller parts of content within a block element. Common inline elements include:

  • <span>
  • <a>
  • <strong>, <em>
  • <img>

Characteristics of inline elements include:

  • Do not start on a new line.
  • Only take up as much width as necessary.
  • Cannot have width and height set via CSS (though padding and margins can be applied).
Example: 
span {
        color: blue; /* Changes text color */
        padding: 5px;
    }

Differences Between Block and Inline Elements

Feature Block Elements Inline Elements
Display Style Starts on a new line Does not start on a new line
Width Takes full width of parent Takes only necessary width
Height Can set width and height Cannot set width and height
Common Elements div, p, h1-h6 span, a, img

Examples and Use Cases

Understanding when to use block and inline elements is crucial for proper layout and styling:

  • Use block elements to structure the main sections of your webpage (e.g., headers, footers, and main content areas).
  • Use inline elements to style smaller pieces of content within block elements (e.g., emphasizing text or adding links).
  • Sometimes, you may need to change the display property of an element using CSS. For example, you can set a block element to display inline using display: inline; or vice versa with display: block;.
Example of changing display: 
div {
        display: inline; /* Changes block to inline */
    }

Conclusion

Understanding the difference between block and inline elements is fundamental in CSS. By mastering this concept, you can create well-structured and visually appealing web pages, ensuring that your content is displayed as intended.
