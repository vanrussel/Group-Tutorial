Lesson 2: Elements & Tags

Introduction

HTML (HyperText Markup Language) defines the structure and content of webpages. We use HTML elements to create all of the paragraphs, headings, lists, images, and links that make up a typical webpage. In this lesson, we will explore how HTML elements work.

Lesson overview

This section contains a general overview of topics that you will learn in this lesson.

  • Explain what HTML tags are.
  • Explain what HTML elements are.

HTML Elements Overview

HTML elements form the structure and content of a webpage. An HTML element typically consists of a start tag, content, and an end tag: 


<tagname>Content goes here...</tagname>

Basic HTML Elements

Here are some common HTML elements:

<h1>This is a Heading</h1>

<p>This is a paragraph.</p>

Start Tag Content End Tag
<h1> Heading Content </h1>
<p> Paragraph Content </p>

Nested HTML Elements

HTML elements can contain other elements. For instance:

<html>
<body>
  <h1>Heading</h1>
  <p>A paragraph inside the body tag.</p>
</body>
</html>

Key HTML Elements

Explore additional HTML elements:

<h3>This is a subheading</h3>

<ul> and <li> for lists:

  • <li>List Item 1</li>
  • <li>List Item 2</li>

Block vs. Inline Elements

HTML elements can be block or inline:

  • Block elements: Always start on a new line and take up full width. Examples: <div>, <p>, <h1>
  • Inline elements: Do not start on a new line. Examples: <span>, <a>
